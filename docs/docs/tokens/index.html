Tokens

Tokens are the source of truth for the UI kit. Components consume them through useTheme() rather than hard-coding color, spacing, or typography values.

Token Layers​

  • colors: primitive color palette and semantic color mappings
  • spacing: spacing scale
  • radius: corner radius scale
  • iconSize: icon sizing scale
  • shadows: elevation presets
  • typography: text styles
  • animation: timing and easing
  • opacity: transparency scale
  • zIndex: stacking layers

Using Tokens​

import { useTheme } from '@zaisustech/ui-kit';

function Example() {
  const { colors, spacing, radius, typography } = useTheme();

  return (
    <View
      style={{
        backgroundColor: colors.background,
        padding: spacing.md,
        borderRadius: radius.md,
      }}
    >
      <Text style={typography.label.bold}>Token-driven content</Text>
    </View>
  );
}

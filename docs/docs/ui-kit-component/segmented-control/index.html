SegmentedControl

A segmented control component for selecting between multiple options.

Category: Molecule
Dependencies: Text, Button

Interactive Preview​

Edit the code to customize the SegmentedControl:

Usage​

import { SegmentedControl } from '@zaisustech/ui-kit';
import { useState } from 'react';

const MyComponent = () => {
  const [selectedValue, setSelectedValue] = useState('first');

  const segments = [
    { label: 'First', value: 'first' },
    { label: 'Second', value: 'second' },
    { label: 'Third', value: 'third' },
  ];

  return (
    <SegmentedControl
      segments={segments}
      value={selectedValue}
      onChange={setSelectedValue}
    />
  );
};

Props​

segments​

Type: Segment<T>[]

Array of segments. Each segment has label: string and value: T.

value​

Type: T

Currently selected segment value.

onChange​

Type: (value: string) => void

Callback fired when segment is selected.

containerStyle​

Type: ViewStyle

Custom styles for the container.

segmentStyle​

Type: ViewStyle

Custom styles for individual segments.

activeSegmentStyle​

Type: ViewStyle

Custom styles for the active segment indicator (the animated background that moves behind the active segment). This can be used to customize the indicator's appearance, including overriding the default borderRadius.

textStyle​

Type: TextStyle

Custom styles for segment text.

activeTextStyle​

Type: TextStyle

Custom styles for active segment text.

Custom Border Radius​

Use activeSegmentStyle to customize the border radius of the indicator:

import { SegmentedControl } from '@zaisustech/ui-kit';
import { useState } from 'react';

function MyComponent() {
  const [selectedValue, setSelectedValue] = useState('first');

  const segments = [
    { label: 'First', value: 'first' },
    { label: 'Second', value: 'second' },
    { label: 'Third', value: 'third' },
  ];

  return (
    <SegmentedControl
      segments={segments}
      value={selectedValue}
      onChange={setSelectedValue}
      activeSegmentStyle={{ borderRadius: 16 }}
    />
  );
}

Custom Container and Indicator Styles​

You can customize both the container and indicator styles:

import { SegmentedControl, defaultTheme } from '@zaisustech/ui-kit';
import { useState } from 'react';

function MyComponent() {
  const [selectedValue, setSelectedValue] = useState('first');

  const segments = [
    { label: 'First', value: 'first' },
    { label: 'Second', value: 'second' },
    { label: 'Third', value: 'third' },
  ];

  return (
    <SegmentedControl
      segments={segments}
      value={selectedValue}
      onChange={setSelectedValue}
      containerStyle={{ backgroundColor: defaultTheme.colors.secondary }}
      activeSegmentStyle={{ 
        borderRadius: 12, 
        backgroundColor: defaultTheme.colors.primary 
      }}
    />
  );
}
